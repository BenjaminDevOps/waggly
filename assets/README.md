# Assets

Ce dossier contient les ressources visuelles de l'application PetHealth.

## Images Requises

Pour que l'application fonctionne correctement, vous devez ajouter les images suivantes :

### Icons
- `icon.png` (1024x1024) - Icône principale de l'application
- `adaptive-icon.png` (1024x1024) - Icône adaptative Android
- `favicon.png` (48x48) - Favicon pour la version web
- `notification-icon.png` (96x96) - Icône des notifications

### Splash Screen
- `splash.png` (1242x2436) - Écran de démarrage

## Générer les Assets

Vous pouvez générer des assets de base avec des outils en ligne ou utiliser les commandes Expo :

```bash
# Générer les icons depuis une image source
npx expo-optimize
```

## Templates

Des templates d'icônes sont disponibles sur :
- [Figma](https://www.figma.com)
- [Canva](https://www.canva.com)
- [Icon Kitchen](https://icon.kitchen)

## Recommandations

- **Format** : PNG avec transparence
- **Résolution** : Haute résolution pour la qualité
- **Style** : Simple et reconnaissable
- **Couleurs** : Cohérentes avec la charte graphique (#6366f1 violet, #22c55e vert)
